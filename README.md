# N-Site-Prototype

For Nestopia homepage design purpose

## 项目概述
这是一个专业的阳光房销售主页原型项目，旨在为Nestopia提供现代化的在线展示平台。

## 主要特性
- 🏠 现代化响应式设计
- 🎨 优雅的渐变色彩搭配
- 📱 完全适配移动端设备
- ⚡ 快速加载优化
- 🔧 易于定制和扩展

## 技术栈
- HTML5 + CSS3 + JavaScript
- Tailwind CSS
- React + TypeScript (可选)
- Vite (构建工具)

## 项目结构
```
├── index.html          # React版本主页
├── sunroom-landing.html # 纯静态版本主页
├── src/               # React源代码
│   ├── components/    # 组件文件夹
│   ├── App.tsx       # 主应用组件
│   └── index.css     # 样式文件
├── public/           # 静态资源
│   └── images/       # 图片资源
└── README.md         # 项目说明
```

## 快速开始
1. 直接打开 `sunroom-landing.html` 查看静态版本
2. 或者运行开发服务器查看React版本

## 部署说明
项目已配置Git版本控制，可直接部署到GitHub Pages或其他静态托管服务。
